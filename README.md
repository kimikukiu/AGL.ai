## Deployment

This project is configured for Fly.io deployment.

### Quick Deploy

Click the button above or visit:
```
https://fly.io/apps/launch?repository=https%3A%2F%2Fgithub.com%2Fkimikukiu%2FAGL.ai
```

### Manual Deploy

```bash
flyctl launch --from=https://github.com/kimikukiu/AGL.ai
```

## Configuration

- `fly.toml` - Fly.io configuration
- `server.js` - Static file server
- Expected URL: `https://agl-ai.fly.dev`
